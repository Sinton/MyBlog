主题真正值得复用的不是某一页，而是这些已经调好气质的零件。你可以新建一个页面，只拿报头、印章和列表；也可以复用 WorkCard，把产品介绍写成一篇案例报道。

这也是它适合作为 GitHub 模板的原因：使用者不必从零设计报纸风格，也不用被一套大而全的主题系统绑住。
